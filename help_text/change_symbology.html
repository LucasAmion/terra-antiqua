How would you like to process the input DEM:
Choose the processing tool you want to use.

This tool sets a new map symbology for the selected DEM.

Raster to be modified:
Select the raster layer you want to change the symbology of.

Color palette:
Choose a color to apply to the selected layer.

Press Add custom color palette button to add a custom color palette to the list of available palettes. Note that currently only color palettes with .cpt extension are supported. The first line in this color palette must start with a '#' sign and have the name of the palette that is displayed in the list of available palettes. cpt-city (.cpt) color palettes can be downloaded from http://soliton.vm.bytemark.co.uk/pub/cpt-city/.
