The (Digital Elevation Models) DEMs often contain artefacts introduced mostly during different processing steps in the framework of this plugin, e.g. while interpolating elevation values to fill in gaps.

Artefact remover is designed to remove them from the DEM raster. The polygons drawn before this dialog emerges, are used to specify the area where artefacts will be removed.
Artefact values are defined by the Expression you enter in the corresponding field.

Your expression should consist of simple comparison operators such as ">", "<", "==" and "within a range". These can be selected from the drop-down menu. You can also use any forms of "NoData" (No data, no data, nodata, NODATA etc) to refer to empty pixels. In this case no pixel will be erased, and the empty pixels will simply be filled with interpolated values.

Field - Enter your expressions:
Enter the expression that will define which range of elevation values is to be consideres as artefacts and thus removed. The expression must contain the letter H (height) which refers to the value of the pixels in the DEM.

Click add more polygons to go to back to the map and draw another polygon. Keep in mind the same expression will be applied to every polygon you draw.

Field - Interpolate values for removed artefacts:
Tick this box to fill in the gaps left by removed pixels.

Warning:
I) Please avoid using these characters in the file name, as they might cause processing errors: ( ) / - % $ @ #
II) If this tool is used repetitively with no path specified, previous results will be overwritten. To avoid this, specify a different path (or filename) each time
III) The polygons you draw while using this tool are temporary and will be erased as soon as the processing is finished.
