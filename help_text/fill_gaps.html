How would you like to process the input DEM:
Choose the processing tool you want to use.

Fill gaps is intended to fill values for gaps between rotated tectonic blocks.

Raster to be modified (Input):
Choose a raster (DEM) layer, which you want to fill gaps in.

Filling type
Choose a filling type. If the "Interpolation type" is selected, it generates elevation data in each empty pixel by interpolating from pixel values around it, using inverse distance weighting. Otherwise a fixed value is filled in.

Filling value
Enter a filling value to fill the empty pixels (NoData values) with. The filling value is applied only to the "Fixed value" filling type.

If you check the box Interpolate inside polygon(s) only, you have to enter a Mask layer and the interpolation will be constrained to the mask(s) from this layer.

Smooth resulting raster:
Checking this box will smooth the resulting raster. This means that the sharp edges will be less pronounced.
If you have checked the box Interpolate inside polygon(s) only, the smoothing will also be constrained to these polygons.
Smoothing type:
Select a Smoothing type: Gaussian filter or Uniform filter. Please refer to the manual for details about these.

Smoothing factor:
You may specify a smoothing factor, which represents the radius around a given pixel that will be taken into consideration. The greater the factor, the smoother will the resulting raster be.

Please be aware that a higher smoothing factor will increase the processing time.

Output file path:
If there is no path specified here, the file will be created in the temporary folder. The full path will be shown in the Log tab and the result will be loaded to the map canvas.

Warning: Please avoid using these characters in the file name, as they might cause processing errors: ( ) / - % $ @ #
Note: If this tool is used repetitively with no path specified, previous results will be overwritten. To avoid this, specify a different path (or filename) each time
