How would you like to process the DEM:
Choose the processing tool you want to use.

Fill gaps is intended to interpolate values for gaps between rotated tectonic blocks. It generates elevation data in each empty pixel by interpolatinf from pixel values around it, using inverse distance weighting.

Raster to be modified (Input):
Choose the raster (topography and bathymetry DEM) where you want to perform the interpolation.
If you check Interpolate inside polygons checkbox, the interpolation will be constrained to the masks you select.

Smooth resulting raster:
Checking this checkbox will smooth the resulting raster. This means that the sharp edges will be less pronounced.
For now two different Smoothing types are available: Gaussian filter and Uniform filter.
You may also specify a smoothing factor, which represents the radius around a given pixel that will be taken into consideration.
The higher the factor, the smoother will the resulting raster be.

Please be aware that smoothing may take 5 to 10 minutes, depending on the system. A higher smoothing factor will increase the processing time.

Specify the Output file path.
If there is no path specified here, the output file will be created in the temporary folder. The full path will be shown in the Log tab and the result will be loaded to the map canvas.

Warnings:
I) Please avoid using these characters in the file name, as they might cause processing errors: ( ) / - % $ @ #
Note:If this tool is used repetitively with no path specified, previous results will be overwritten. To avoid this, specify a different file path each time
