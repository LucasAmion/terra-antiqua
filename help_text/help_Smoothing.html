This tool is designed to reduce abrupt topography changes in a DEM. It is meant to be used while paleoreconstruction after interpolation or topography modification. However, it may be used in any topography raster.

Field - Select the compiled topography+bathymetry raster
You need to specify the raster where you want to perform the smoothing.

Field - Soothing factor
Enter here the number of cells that should be taken into account to calculate the smoothing.
