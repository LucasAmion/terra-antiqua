How would you like to process the DEM:
Choose the processing tool you want to use.

This tool calulates bathymetry given present day ocean age, using the following equation:
Od = -2620 -330 * √Oa

Where:
Od - Ocean depth
Oa - Ocean age

Raster to be modified
Select a raster layer, Ocean age grid that is rotated to the time of reconstruction.

The Reconstruction time parameter defines the amount of the sea level vertical shift relative to its present day level. For example, if the sea level should be dropped by 100 meters this parameter should set a value of -100 m. In this case the topography will get higher by 100 m and the bathymetry will get shallower by this same amount.

Specify the Output file path.
If there is no path specified here, the file will be created in the temporary folder. The full path will be shown in the Log tab and the result will be loaded to the map canvas.

Warnings:
I) Please avoid using these characters in the file name, as they might cause processing errors: ( ) / - % $ @ #
Note:If this tool is used repetitively with no path specified, previous results will be overwritten. To avoid this, specify a different path (or filename) each time
