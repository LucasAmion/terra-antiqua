This tool integrates the paleoshorelines into the DEM compiling the topography and bathymetry. Based on the mask provided, it will be checked if regions that should be emerged appear underwater and if submerged regions are shown above the sea level. The resulting raster will include these corrections by giving the value 1 to the newly emerged areas and -100 to the newly submerged areas.

Field - Topography raster:
Select the raster compiling the topography and the bathymetry (rotated to the time of reconstruction)

Field - Shorelines:
Select the vector layer with the paleoshorelines (rotated to the time of reconstruction).

Select the output file path.
