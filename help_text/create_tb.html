This tool creates new topography or bathymetry in a DEM in the area defined by a polygon. For details about the process, please refer to the manual.

Raster to be modified:
Select a raster layer in which you want to create the features.

Layer with feature polygons:
Select a vector layer containing the polygons where you want to create new geographic features.
If only certain masks should be used to create the feature(s), these masks must be selected and Selected feature(s) only must be checked.

Geographic feature type:
Select the type of the geographic feature that you want to create (sea, mountain range).

Note: All the numerical parameters that follow (max depth, min depth, etc.) can be selected from the attribute table of the Layer with feature polygons. For this click on the button on the right of each line to select the corresponding field of the attribute table.

Maximum sea depth / ridge elevation (in m):
Enter here the maximum elevation (for topography) or maximum depth (for bathymetry) that the new features must have.

Minimum sea depth / ridge elevation (in m):
Enter here the minimum elevation (for topography) or minimum depth (for bathymetry) that the new features must have.

Ruggedness of the mountains (in %): (only available for mountains)
The lower this value, the "smoother" will the mountain be.

Shelf width (in km): (only available for seas)
Enter here the width of the shallow area that will be created at the margins of the sea (continental shelf).

Width of the continental slope: (only available for seas)
Enter here how large you want to make the contiental slope, which will start at the end of the shelf you define above. The higher this number is, the flatter will the slope be.

Width of the mountain slope: (only available for mountains)
Enter here how large you want to make the mountain slope, which starts from the edge of the polygon and goes towards the center. The higher this number is, the flatter will the slope be.

Advanced parameters:
Color palette:
Here you may choose between the color schemes stored in the corresponding folder in Plugins/Terra Antiqua. By default the Terra Antiqua color scheme is selected. Please refer to the manual for instructions about downloading and saving color palettes.

Output file path: If there is no path specified here, the file will be created in the temporary folder. The full path will be shown in the Log tab and the result will be loaded to the map canvas.

Warning: Please avoid using these characters in the file name, as they might cause processing errors: ( ) / - % $ @ #
Note: If this tool is used repetitively with no path specified, previous results will be overwritten. To avoid this, specify a different path (or filename) each time
