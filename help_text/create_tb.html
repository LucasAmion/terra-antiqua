This tool creates new topography or bathymetry in a DEM in the area defined by a polygon.

Geographic feature creation


Field - Geographic feature type:
Select whether you want to create an ocean or a mountain range. For details about the way these are created, please refer to the manual. Field - Raster to be modified:
Select the raster where you want to create the features.

Field - Layer with polygons for the geographic features:
Select the vector file containing the polygons where you want to create new geographic features.

Field - Maximum sea depth / ridge elevation (in m):
Enter here the maximum elevation (for topography) or depth (for bathymetry) that the new features must have.

Field - Minimum sea depth / ridge elevation (in m):
Enter here the minimum elevation (for topography) or depth (for bathymetry) that the new features must have.

Field - Ruggedness of the mountains (in %):
(only available for mountains) The lower this percentage, the "smoother" will the mountain be.

Field - Shelf width (in km):
(only available for seas) Enter here the width of the shallow sea area that will be created at the margins of the sea.

Field - Width of the continental slope:
(only available for seas) Enter here how large you want to make the contiental slope, which will start at the end of the shelf you define above. The higher this number is, the flatter will the slope be.

Field - Width of the mountain slope:
(only available for mountains) Enter here how large you want to make the mountain slope, which will start from the edge of the polygon. The higher this number is, the flatter will the slope be.

Specify the Output file path.
If there is not a path specified here, the file will be created in the temporary folder. The full path will be shown in the Log tab and the result will be loaded to the map canvas.

Warnings:
I) Please avoid using these characters in the file name, as they might cause processing errors: ( ) / - % $ @ #
II) If this tool is used repetitively with no path specified, previous results will be overwritten. To avoid this, specify a different path (or filename) each time
