This tool creates new topography or bathymetry in a DEM in the area defined by a polygon.

Raster to be modified:
Select a raster layer, in which you want to create the features.

Layer with feature polygons:
Select a vector layer containing the polygons where you want to create new geographic features. If only certain masks should be used for feature creation, these masks must be selected and then the Selected features only must be checked.

Geographic feature type::
Select the type of the geographic feature that you want to create (ocean, mountain range).

Maximum sea depth / ridge elevation (in m):
Enter here the maximum elevation (for topography) or depth (for bathymetry) that the new features must have.
Note: All the parameters defining the geometry of the feature to be created (max depth, min depth etc.) can be selected from the attribute table (if the parameters are different for each feature). To this end, a field should be specified in the menu by pressing the button on the right side of each field.

Minimum sea depth / ridge elevation (in m):
Enter here the minimum elevation (for topography) or depth (for bathymetry) that the new features must have.

Ruggedness of the mountains (in %):
(only available for mountains) The lower this value, the "smoother" will the mountain be.

Shelf width (in km):
(only available for seas) Enter here the width of the shallow sea area that will be created at the margins of the sea (continental shelf).

Width of the continental slope:
(only available for seas) Enter here how large you want to make the contiental slope, which will start at the end of the shelf you define above. The higher this number is, the flatter will the slope be.

Width of the mountain slope:
(only available for mountains) Enter here how large you want to make the mountain slope, which will start from the edge of the polygon (toward the center). The higher this number is, the flatter will the slope be.

Specify the Output file path.
If there is no path specified here, the file will be created in the temporary folder. The full path will be shown in the Log tab and the result will be loaded to the map canvas.

Warnings:
I) Please avoid using these characters in the file name, as they might cause processing errors: ( ) / - % $ @ #
Note:If this tool is used repetitively with no path specified, previous results will be overwritten. To avoid this, specify a different path (or filename) each time
