/***********************************************************************
Terra Antiqua
A QGIS plugin

The Terra Antiqua plugin creates a paleogeographic map of a specific time. It modifies present day topography and bathymetry that is rotated to the time of reconstruction in Gplates with a set of masks that are also rotated in Gplates.

terra_antiqua.core.utils.randomPointsInPolygon(source, point_density, min_distance, feedback, runtime_percentage)¶

Creates random points inside polygons.

Parameters:
  • source – input vector layer with polygon features, inside which the random points will be created.
  • point_density – the density of points to be created inside polygons. Number of points to be created will be calculated with the following formula: Pnumber = Pdensity * PolygonArea.
  • min_distance – Minimum distance that will be kept between created points.
  • feedback – a feedback object to provide progress and other info to user. For now a Qthread object is passed to use its pyqtsignals, functions and attributes for feedback purposes.
  • runtime_percentage – time that this part of the algorithm will take (this function is run inside an algorithm e.g. Feature Creator) in percent (e.g. 10%)

Rasterizes a vector layer and returns a numpy array.

Parameters:
  • field_to_burn – A specific field from attributes table to get values to burn. This can be a field with depth or elevation values.
  • no_data – No data value. It can be NAN, zero or any other value.
  • burn_value – A fixed value to burn in the raster.
  • geotransform – geotransform for the resulting raster layer. Can accept geotransform (raster_ds.GetGeotransform()) extent (raster_layer.extent()) and QgsRasterLayer.
  • width – number of columns in the raster. Should be consistent with the raster that the masks will deployed on.
  • height – number of rows in the raster. Should be consistent with the raster that the masks will deployed on.

Numpy array.
